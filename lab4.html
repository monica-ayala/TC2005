  • ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

    Son lenguajes de programación completamente distintos, cuya única similitud se basa en que fueron creados a partir del lenguaje C. Java es un lenguaje compilado, JavaScript es un lenguaje interpretado. Java es puramente un lenguaje Orientado a Objetos y JavaScript está orientado a prototipos.

  • ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

    Al declarar una variable global está podrá ser accesada por todo el código y toda las funciones del mismo, al usar las variables globales dentro de las funciones las sometes a posibles cambios y a usar el último valor asignado/modificado de la variable, no necesariamente aquel que fue declarado inicialmente.
